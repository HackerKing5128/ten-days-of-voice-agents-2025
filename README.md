## ğŸš€ Quick Start

### Prerequisites

- Python 3.9+ with [uv](https://docs.astral.sh/uv/)
- Node.js 18+ with pnpm
- [LiveKit Server](https://docs.livekit.io/home/self-hosting/local/)

### Run Any Day's Agent

```bash
# Clone the repo
git clone https://github.com/HackerKing5128/ten-days-of-voice-agents-2025.git
cd ten-days-of-voice-agents-2025

# Switch to any day's branch (e.g., Day 10)
git checkout day10-voice-improv

# Setup backend
cd backend
uv sync
cp .env.example .env.local
# Edit .env.local with your API keys

# Setup frontend
cd ../frontend
pnpm install
cp .env.example .env.local

# Run (3 terminals)
# Terminal 1: ./livekit-server.exe --dev
# Terminal 2: cd backend && uv run python src/agent.py dev
# Terminal 3: cd frontend && pnpm dev

# Open http://localhost:3000
```

### Required API Keys

```env
LIVEKIT_URL=ws://localhost:7880
LIVEKIT_API_KEY=devkey
LIVEKIT_API_SECRET=secret
MURF_API_KEY=your_murf_api_key
GOOGLE_API_KEY=your_google_api_key
DEEPGRAM_API_KEY=your_deepgram_api_key
```
